================================================================================
                    HANDORRA - HOW TO RUN THE PROJECT
================================================================================

STEP 1: INSTALL DEPENDENCIES
-----------------------------
Open your terminal in the Handorra-2.0-main folder and run:

    npm install

This will install all required packages (React, TypeScript, Tailwind, etc.)


STEP 2: CONFIGURE ENVIRONMENT
------------------------------
The .env file is already set up with Supabase credentials.

If you need to use your own Supabase project:
1. Go to https://supabase.com
2. Create a new project
3. Get your credentials from Settings > API
4. Update the .env file with your values


STEP 3: RUN THE APPLICATION
----------------------------
Start the development server:

    npm run dev

The app will start on: http://localhost:8080


STEP 4: OPEN IN BROWSER
------------------------
Navigate to:

    http://localhost:8080

You should see the Handorra homepage!

================================================================================
                              TROUBLESHOOTING
================================================================================

Problem: Port 8080 already in use
Solution: Edit vite.config.ts and change the port number

Problem: Cannot connect to Supabase
Solution: Check your .env file has correct credentials

Problem: Module errors
Solution: Delete node_modules and package-lock.json, then run npm install
